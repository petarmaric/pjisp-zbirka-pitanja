Listanje sadrzaja direktorijuma u terminalu se vrsi putem:
- mkdir
- cd
- dir
+ ls
.

Promena direktorijuma u terminalu se vrsi upotrebom komande:
- mkdir
+ cd
- dir
- ls
.

Prelazak u roditeljski direktorijum u terminalu se vrsi upotrebom komande:
- mkdir ..
- cd
- cd .
+ cd ..
- cd ~
- cd ./
- dir ~
- ls .
.

Kompajliranje u terminalu upotrebom GNU Compiler-a se vrsi putem:
- gcc -g -o zadatak.c zadatak
- gcc -o zadatak.c zadatak
- cc -g -o zadatak zadatak.c
- cc -o zadatak zadatak.c -lm
- gcc -o zadatak zadatak
+ gcc -o zadatak zadatak.c
.

Kompajliranje u terminalu upotrebom GNU Compiler-a sa upotrebom matematicke biblioteke se vrsi putem:
- gcc -o zadatak.c zadatak -lm
- gcc -o zadatak.c zadatak
- cc -g -o zadatak zadatak.c
- cc -o zadatak zadatak.c -lm
+ gcc -o zadatak zadatak.c -lm
.

U Linux terminalu, nakon poziva komande 'gcc -o zadatak zadatak.c', dobijena izvrsna datoteka se pokrece komandom:
- zadatak
- /zadatak
- a.out
- ./a.out
- ./zadatak.c
+ ./zadatak
.
